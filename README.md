🕹️ Jogo da Velha (Tic-Tac-Toe CLI)
Projeto interativo de Jogo da Velha desenvolvido em Python para rodar diretamente no terminal. O foco deste trabalho foi aplicar conceitos fundamentais de lógica de programação e manipulação de dados em uma interface CLI (Command Line Interface).

🚀 Tecnologias e Conceitos
Linguagem: Python 3.
Estruturas de Dados: Uso de listas para representação e atualização do tabuleiro.
Controle de Fluxo: Implementação de laços while para o ciclo de jogo e condicionais if/else para validação de regras.
Interface: Estilização do terminal com cores ANSI para melhor experiência do usuário.
✅ Funcionalidades
Alternância de jogadores: Troca automática entre 'X' e 'O' a cada rodada.
Verificação de vitória e empate: Algoritmo que checa linhas, colunas e diagonais.
Validação de jogadas: Impede a escolha de posições já ocupadas ou inválidas.
Opção de Reinício: Sistema que permite iniciar uma nova partida sem encerrar o programa.
▶️ Como executar
Certifique-se de ter o Python instalado.
Salve o código como jogo_da_velha.py.
No terminal, execute:
python jogo_da_velha.py
